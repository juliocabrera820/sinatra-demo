# Demo Sinatra 

Ruby version - 2.5.7

## Getting started

`bundle install`

## Database creation

`rake db:create`

`rake db:migrate`

## Development

After checking out the repo, run `bundle install` to install dependencies. Then run `shotgun` to run sinatra app and copy
the next url in browser `http://localhost:9393/`
